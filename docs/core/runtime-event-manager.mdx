`RuntimeEventManager` stores all runtime event definitions registered via `action.registerRuntimeEventClass()`. It
extends [`BaseManager`](./base-manager.mdx) with key type `string` (identifier) and value type `RuntimeEventProps`.

```ts
action.runtimeEvents.get('user_created');
action.runtimeEvents.values();
```

## Interfaces

### `RuntimeEventRunnable`

An empty marker interface. RuntimeEvent classes do not need to implement any methods.

```ts
interface RuntimeEventRunnable {}
```

### `RuntimeEventProps`

The descriptor stored for each registered runtime event.

```ts
interface RuntimeEventProps {
  identifier: string;
  settings?: EventSettingProps[];
  signature: string;
  editable?: boolean;
  name?: Translation[];
  description?: Translation[];
  documentation?: Translation[];
  displayMessage?: Translation[];
  alias?: Translation[];
  displayIcon?: string;
}
```
